# TDownload

Package tiktok downloader

## Installation

You can install the package via composer:

```bash
composer require zhxdev/tdownload
```

## Usage

```php
use Zhxdev\TDownload\TDownload;

$packageDetail = (new TDownload)->getPackageDetail();

var_dump($packageDetail);
```

## Testing

```bash
composer test
```
